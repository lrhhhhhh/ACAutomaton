# 题目
`https://leetcode-cn.com/problems/add-strings/`


# 思路
又想骗我写大数加法


```python
class Solution(object):
    def addStrings(self, num1, num2):
        """
        :type num1: str
        :type num2: str
        :rtype: str
        """
        one = int(num1)
        two = int(num2)
        return str(one+two)
```
